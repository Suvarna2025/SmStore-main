### Installation

- [ ] Clone the Repository  

```
    $ git clone https://github.com/nathan-somto/smstore.git
```

- [ ] Enter into the directory

```
    $ cd Smstore
```

- [ ] Install all dependencies

```
   $ npm install
```

- [ ]  Start the Development Server

```
    $ npm run dev
```
